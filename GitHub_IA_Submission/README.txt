In order for the product to run all the dependencies listed in the requirements.txt need to be installed. Furthermore, a MYSql server needs to be set up and a database created. In order to do that a simple tutorial it's sufficient. Once the server is running,  open the file __init__.py inside the myproject folder. In there you need to replace db_manager with the name of your server, ciao with the password of your server, and db_booking with the name of your created database. Once you did that you only need to run the app.py document from the command line terminal.
